# StoryVerse - Where Stories Come Alive

An interactive literary platform that transforms static books into living, interactive worlds powered by AI.

## Quick Start

### Local Development
1. Clone this repository
2. Serve over HTTP (required for data loading):
   ```bash
   npx serve .
   ```
3. Open http://localhost:3000 in your browser
4. No build process needed - it's a static site!

### Deploy to Vercel
1. Push to GitHub
2. Import repository in Vercel
3. Deploy - no configuration needed
4. Done!

## Features

- **Landing Page**: Hero section, features, pricing tiers
- **Library**: Browse interactive book worlds
- **Character Chat**: AI-powered conversations with book characters
- **Author Dashboard**:
  - Manuscript upload simulation
  - Character profile management
  - Interaction parameter controls
  - Analytics dashboard with charts

## Business Model

**Reader Subscriptions**
- Free: $0/mo (20 messages)
- Explorer: $9.99/mo (unlimited)
- Creator: $19.99/mo (all features)

**Author Subscriptions**
- Indie: $29/mo (1-3 books, 70% revenue share)
- Professional: $79/mo (10 books, 75% revenue share)
